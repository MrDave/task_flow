# TaskFlow
A full-stack task management platform with a Django REST API.

## Description
TaskFlow is a task manager app that allows user to create and manage personal tasks in a simple way.
It uses Django and Django REST Framework. Frontend TBD.

## Project structure

```
/
├── backend/   – Django + DRF API, JWT Auth, PostgreSQL
├── frontend/  – Client application (framework TBD)
└── README.md  – You are here
```

## Tech stack
### Backend:
- Python 3.x
- Django
- Django REST Framework
- SimpleJWT (JWT Authentication)
- PostgreSQL database
